# minesweeper-bot

bot that solves minesweeper

[link to game](https://alegoritmia.github.io/minesweeper-bot/)

## Current progress

working minesweeper game with default difficulties

## Controls/How to play

- right click for clicking on cells and show them
- left click to mark/unmark flags
- flags are unclickable
- right click on uncovered number with correct amount of flags to show surrounding cells
